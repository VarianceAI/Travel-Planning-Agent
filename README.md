# ✈️ Travel Planning AI Agent

## 🚀 Overview

This project is an AI-powered autonomous travel planning agent
that analyzes user intent and dynamically compares flight and hotel prices
across multiple platforms.

## 🏗 Architecture

Frontend: Next.js  
Backend: FastAPI  
LLM Layer: OpenAI GPT-4  
Cache: Redis  
Deployment: Docker + GCP Cloud Run  

## ⚡ Core Capabilities

- Intent parsing using LLM
- Real-time price aggregation
- Itinerary generation
- Budget optimization

## 🧠 System Design Highlights

- Stateless API design
- Redis caching layer
- Async API calls for parallel pricing fetch
- Containerized deployment
